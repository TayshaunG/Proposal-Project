## Setup & Running

### 1. Backend
```bash
cd backend
pip install -r requirements.txt
python train.py        # generates synthetic data + trains models
python run.py          # starts Flask API on http://localhost:5000
```

### 2. Frontend
```bash
cd frontend
npm install
npm run dev            # starts React app on http://localhost:3000
```

Then open **http://localhost:3000** in your browser.

## API Endpoints
| Method | Endpoint | Description |
|--------|----------|-------------|
| GET  | /api/health | Check API is running |
| POST | /api/classify | Classify a single flow |
| POST | /api/classify/batch | Classify multiple flows |
| GET  | /api/history | Get classification history |
| DELETE | /api/history | Clear history |
| GET  | /api/stats | Model metrics + session stats |
| GET  | /api/features | List of expected feature names |

### Example classify request:
```json
POST /api/classify
{
  "named_features": {
    "Flow Duration": 500000,
    "Total Fwd Packets": 10,
    "Flow Bytes/s": 15000,
    "SYN Flag Count": 1
  }
}
```

## Models
- **Isolation Forest** — unsupervised, detects anomalies without labels
- **Random Forest** — supervised classifier trained on labelled data
- Both run on every request; consensus logic determines the final label

## Dataset
- Development: synthetic CICIDS2017-like data (generated by `train.py`)
- Final evaluation: real CICIDS2017 dataset from https://www.unb.ca/cic/datasets/ids-2017.html
